
# Image Compression Tool

A Python-based image compression tool built using Gradio and Pillow.

## Features
- Upload image and compress it
- Adjust compression level
- View original vs compressed size
- Download compressed image

## Technologies Used
- Python
- Gradio
- Pillow (PIL)

## How to Run
1. Install dependencies:
   pip install gradio pillow

2. Run the script:
   python app.py
